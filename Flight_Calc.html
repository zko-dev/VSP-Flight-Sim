VSP Flight Sim - Trim state convergence study¶

Readme: This code is used in pair with OPENVSP to generate a rough estimate to the performance of Group 1 Fixed Wing UAV under steady state scenarios.

Steps¶

  1. If initial run, do setup.sh to install/update requirements.txt
  2. From CAD model, determine MTOW, CG into this Flight_Calc.ipynb
    • Modify existing .vspaero file to setup for cfd
  3. run_sim.sh
    • launches openvsp GUI
    • Generate degenGeom from .vsp3 using launch_openvsp.sh
    • runs batch sweep and triggers vspaero solver
    • outputs vsp_aero_result.csv in /output
  4. Flight_Calc.ipynb extracts vsp_aero_result.csv into custom result display
    • Runs propulsion trade study

import numpy as np

# === Functions ===
def air_density_at_altitude(h):
    T0 = 288.15
    P0 = 101325
    L = 0.0065
    R = 287.058
    g = 9.80665
    T = T0 - L * h
    P = P0 * (1 - L * h / T0) ** (g / (R * L))
    return P / (R * T)

# === Conditions ===

# === Thrust using disk advancing ratio model ===
def estimate_thrust(V, RPM, D, CT, pitch):
    n = RPM / 60
    V_pitch = pitch * 0.0254 * n
    T_static = CT * rho * n**2 * D**4
    T = T_static * (1 - V / V_pitch)**2
    T[V >= V_pitch] = 0
    return T
T_avail = estimate_thrust(V, RPM_max, D_m, CT, pitch_in) *n_forward_rotors
def thrust_diff(v):
    cl = (2 * W) / (rho * v**2 * Sref)
    cd = CD0 + k * cl**2
    t_req = 0.5 * rho * v**2 * Sref * cd
    V_pitch_local = pitch_in * 0.0254 * (RPM_max / 60)
    T_static = CT * rho * (RPM_max / 60)**2 * D_m**4
    T_av = n_forward_rotors*T_static * (1 - v / V_pitch_local)**2 if v < V_pitch_local else 0
    return T_av - t_req

# === Model: Thrust from Current (reversed momentum theory) ===
def thrust_from_current(I_motor, D_m, V_batt, rho, pitch_in, p0, alpha_hover,
                        eta_prop=0.7, eta_motor=0.85, I_0=0.8):
    I_base = I_motor / (1 + alpha_hover * (pitch_in - p0))
    P_elec = (I_base - I_0) * V_batt
    A = np.pi * (D_m**2) / 4
    P_ideal = P_elec * eta_prop * eta_motor
    T = (P_ideal * np.sqrt(2 * rho * A))**(2/3)
    return T

# === Hovering Subsection (Momentum + Pitch Correction) ===
def hover_current_momentum(thrust_N, D_m, V_batt, rho, pitch_in, p0, alpha_hover,
                           eta_prop, eta_motor, I_0=0.8):
    A = np.pi * (D_m**2) / 4
    P_ideal = thrust_N**1.5 / np.sqrt(2 * rho * A)        # Ideal induced power
    P_elec = P_ideal / (eta_prop * eta_motor)             # Electrical power input
    I_base = P_elec / V_batt + I_0                        # Base current estimate
    pitch_correction = 1 + alpha_hover * (pitch_in - p0)  # Correction based on pitch
    return I_base * pitch_correction
